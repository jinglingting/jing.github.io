MySQL碎片整理

Posted on 2019-03-02 | Edited on 2019-03-10 | In MySQL优化

存储引擎为MyISAM, 定期碎片整理

  • 对表存储引擎为MyISAM进行整理, 使用 optimize table table_name;
  • 对表存储引擎为InnoDB进行整理, 使用 alter table table_name engine=innodb;
    • 对于存储引擎为InnoDb的表整理, 也可以使用 optimize table table_name, 只是会映射为alter table ...的命令
